# localhost-tunneling

A basic client/server up to implement localhost tunneling


## .envrc example

```bash
export CF_API_TOKEN=

export DOMAIN=
export CADDY_ADMIN_API=localhost:2019
export TUNNEL_SERVER_PORT=4000
export CADDY_PROXY_PORT=3000

export EMAIL=
export PRODUCTION=true

# Client variables
export TUNNEL_SERVER_IP=
```
